def fit_curve(tempo, empuxo):
    def objective(x, a, b, c, d, e, f):
        return (a * x) + (b * x**2) + (c * x**3) + (d * x**4) + (e * x**5) + f

    popt, _ = curve_fit(objective, tempo, empuxo)
    return popt


def generate_curve(popt, x_values):
    a, b, c, d, e, f = popt
    return [(a * x) + (b * x**2) + (c * x**3) + (d * x**4) + (e * x**5) + f for x in x_values]

Informaes gerais

O voo do foguete pode ser dividido em 3 fases.

  • Propulsada
  • Balstica
  • Sustentada

Em cada uma das fases, diferentes foras esto em ao.

  • $T$: Fora de empuxo, proveniente da queima do propelente.
  • $D$: Fora de resistncia aerodinmica, pode ser pelo foguete ou pelo paraquedas.
  • $P$: Fora peso, muda de acordo com o consumo do propelente.

O interesse definir a posio da altura no tempo $h(t)$ com foco no apogeu atingido. De maneira geral:

  • $h(t)$ - Posio
  • $\dot h(t) = V$ - Velocidade
  • $\dot V = \sum F/m$ - Acelerao

Nesse estudo, o movimento considerado em apenas em uma direo.

Fase propulsada

Equao para $\dot V$:

$\dot V = [-(P(m) + D(V)) + T(t)]/m$

Onde:

  • A massa m uma funo $m(t)$;
  • O empuxo T uma funo $T(t)$;
  • D uma funo $D(V)$

Equaes envolvidas:

# Variveis tempo, gravidade, massa, massa do propelente, massa seca,
t, g, m, mp, ms, p, c, a, tq = sp.symbols('t g m mp ms p c a tq')
# densidade do meio (ar), coeficiente de arrasto, rea de referncia, tempo de queima

h = sp.symbols(r'h', cls=sp.Function)
h = h(t)  # Altura
h_dot = sp.diff(h, t)  # dh/dt (Velocidade)
h_ddot = sp.diff(h_dot, t)  # dh/dt (Acelerao)

Definio da acelerao ou $\dot V$ considerando o empuxo e massa variveis com o tempo e a resistncia aerodinmica varivel com a velocidade:

In[754]:
mt = mp + ms  # Massa total
m = mt - ((mp/tq)*t)  # Funo da massa
P = m * g  # Fora peso

D = 0.5 * p * h_dot**2 * c * a  # Fora de arrasto # Empuxo

T = sp.sympify(curva) # Empuxo

vdot = (-P - D + T)/m

dvdt = sp.Eq(vdot, h_ddot)  # Equao da acelerao
sol = sp.solve([dvdt], [h_ddot])

dvdt.simplify()

ddhddt_f = sp.lambdify((h_dot, t, g, mp, ms, p, c, a, tq), sol[h_ddot]) # Funo da acelerao 
dhdt_f = sp.lambdify(h_dot, h_dot) # Funo da velocidade 
pesot = sp.lambdify((t, mp, ms, tq, g), P) # Funo da massa total

def dSdt(S, t, g, mprop, mseco, ro, cd, af, tq):
    h, hd = S

    return [dhdt_f(hd),
            ddhddt_f(hd, t, g, mprop, mseco, ro, cd, af, tq)]

Fase balstica

No h mais a ao de fora do motor nem a variao de massa.

$\dot V = -(P + D(V))/m$

Onde:

  • A massa m constante;
  • O empuxo T no atua mais;
  • D uma funo $D(V)$

A resistncia aerodinmica relacionada ao dimetro do foguete e contrria ao seu movimento.

Definio da acelerao ou $\dot V$:

P = m * g # Fora peso constante

vdot = -(P + D)/m

Fase sustentada

Fase em que o paraquedas acionado para sustentar a queda do foguete sem danos ao aterrisar.

Mantem-se as variveis anteriores, porm o Coeficiente de Arrasto e rea de Referncia considerados so do paraquedas.

A resistncia aerodinmica se torna fora contrria fora peso.

$\dot V = (-P + D(V))/m$

Onde:

  • A massa m constante;
  • O empuxo T no atua mais;
  • D uma funo $D(V)$

Obteno da acelerao ou $\dot V$:

In[762]:
vdot = (-P + D)/m

dvdt = sp.Eq(vdot, h_ddot) # Equao da acelerao
sol = sp.solve([dvdt],[h_ddot])

dvdt.simplify()

Sistemas de equaes solucionveis:

ddhddt_f = sp.lambdify((h_dot, g, ms, p, c, a), sol[h_ddot]) # Funo da acelerao

# Sistema de equaes
def dSdt(S, t, g, mseco, ro, cd, af):
    h, hd = S

    if h <= 0:
        return [0, 0]
    else:
        return [dhdt_f(hd),
                ddhddt_f(hd, g, mseco, ro, cd, af)]
